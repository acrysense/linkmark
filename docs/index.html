Добро пожаловать!

Вы находитесь в мобильной версии сервиса поиска товарных знаков LINKMARK Lite с ограниченным функционалом. Для использования всех возможностей поиска, пожалуйста, перейдите на полную версию сайта с компьютера — https://linkmark.ru/

В LINKMARK Lite доступны следующие опции поиска по зарегистрированным товарным знакам:

  • По названию;
  • По номеру;
  • С учетом классов МКТУ.

Поиск по зарегистрированным объектам абсолютно бесплатный!

В полной версии сайта LINKMARK для ПК доступен расширенный функционал поиска:

  • По изображениям;
  • По правообладателям;
  • По заявкам;
  • По Международным непатентованным наименованиям (МНН);
  • По статусу (действует/не действует/продлен).

Вы можете задавать удобные дополнительные фильтры для более тонких настроек выдачи результата.

В версии для ПК для вас доступны удобные функции добавления объектов в мониторинг, хранения истории поисковых запросов, сохранения и распечатки полученных результатов.

Информация в базах данных обновляется один раз в неделю.
